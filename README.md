# Vco Riot Mobile 维酷移动框架  #

vcotime.com 's webapp framework for mobiles

## 框架集成： ##

`fastclick.js` 移动端快速点击框架

`riot.js` 1kb MVP 开发框架

`zepto` jquery 移动框架

`bouncefix.js` IOS 修复拖动bug框架

`boilerplate` html5 resset
